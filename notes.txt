Pseudocode:

for i from 0 to n-2 do:
    for j from 0 to n-2-i do:
        if list[j] > list[j+1]:
		    swap list[j], list[j+1]
		
			
			
BCE O(n2)
WCE O(n2)


Pseudocode:


for i from 0 to n-2 do:
    sorted = true # assume the list is sorted
    for j from 0 to n-2-i do:
        if list[j] > list[j+1]:
		    swap list[j], list[j+1]
			sorted = false
	if sorted:
	    break

BCE O(n)
WCE O(n2)
